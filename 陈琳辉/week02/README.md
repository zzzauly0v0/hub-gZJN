# Week02 - 多分类任务

## 任务描述

使用全连接神经网络（MLP）对随机向量进行多分类：
- **输入**：一个随机向量（如 `[0.1, 0.8, 0.3, 0.2, 0.4]`）
- **标签**：向量中最大值的索引（如上例中最大值 0.8 在索引 1，标签为 1）
- **类别数**：等于向量维度（默认 5 类）

## 文件说明

| 文件 | 说明 |
|------|------|
| `multiclass_train.py` | 主训练脚本，包含模型定义、数据集、训练和可视化函数 |
| `inference_test.py` | 推理测试脚本，加载模型进行预测 |
| `requirements.txt` | 依赖包 |

## 模型细节

### 网络结构

```
MultiClassClassifier(
  (net): Sequential(
    (0): Linear(5 -> 64)      # 输入层 -> 隐藏层
    (1): ReLU()               # 激活函数
    (2): Linear(64 -> 64)     # 隐藏层
    (3): ReLU()               # 激活函数
    (4): Linear(64 -> 5)      # 隐藏层 -> 输出层
  )
)
```

- **输入层**：5 维（向量维度）
- **隐藏层**：64 维，ReLU 激活
- **输出层**：5 维（类别数）
- **参数总数**：约 960 个

### 训练配置

- **损失函数**：CrossEntropyLoss（内置 softmax）
- **优化器**：Adam（lr=0.001）
- **批次大小**：32
- **训练轮次**：100
- **训练样本**：1000，测试样本：200

### 设备支持

自动检测并优先使用 MPS（Apple Silicon GPU），无则回退到 CPU。
